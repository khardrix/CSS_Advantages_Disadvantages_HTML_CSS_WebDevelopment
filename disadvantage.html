Disadvantages of CSS

  • Lack of security
  • What works on one browser, might not work on another browser
  • CSS comes in different levels (CSS1, CSS2, CSS3), which has resulted in confusion in developers and browsers
  • A webpage will take longer to load if CSS is embedded in it
